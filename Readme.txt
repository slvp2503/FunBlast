# 🎮 Fun Blast

A 2-player anime battle game built using Python and Pygame.

## 🔥 Features
- 2-player local multiplayer
- Character selection (Goku, Vegeta, Violet Ever Garden)
- Directional attacks (can change direction mid-attack)
- Transformations with cooldown & invincibility
- Score system and win condition
- Sound effects and background music

## 🎮 Controls

### Player 1
- Move: Arrow keys
- Attack: Numpad (1,2,3,5)
- Transform: 4

### Player 2
- Move: WASD
- Attack: I, J, K, L
- Transform: U

## 🚀 How to Run

### Option 1 (Recommended)
Just download the exe file and enjoy ! ( no need to download anything else )
( if you want the code I posted a text document which you can directly download )

### Option 2
Install Python
Go to command prompt and paste " pip install pygame "
and then run main.py and enjoy !


## 📌 Tech Used
- Python  
- Pygame
